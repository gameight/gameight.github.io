About The Game


The project is a mobile game which tests your skills in various ways. Type of game action, platfrom and the game is 2D.The game will consist of levels.

We have a character that called Rina. Game story begins with the kidnapping of Rina's sister by the Orcs. At every level, our character is fighting enemies that kidnapped her sister. If the player can cross all levels, she has the opportunity to fight the biggest orc and save her sister.

System Requirements


Functional Requirements
Include multiple levels with a scenario.
Levels will be sorted from easy to hard.
Game include tutorial level to teach some functions to player.
Each level include hidden part to fight bosses and taking skills from them.
Player must choose two different combinations of current skills to kill the boss.
Each level include moving platforms,orcs,boss and various traps.
Each bosses and orcs have different attack type.
When Rina takes damage, her health is decrement.
Each level include check point that Rina returns the midpoint of the level.
When her health is zero, Rina goes the Check point.
When her remaining lives is zero, game over.
Player can play the previously completed levels but cannot play the unlocked levels.
Player can see which level locked or not in the map scene.
Menu include sound and language settings.
There will be an interface to display Google Play Game Services.
Sign in will be automatic.

Non-Functional Requirements
Performance: We will build the game in Unity, which is one of the best 2D game engines. Our game will be optimized for scripts, garbage collector and graphic rendering after building the raw game. These conditions will provide high performance for our users.

Scalability: Our game will be accessible for all Play Store users. There is no user limitation to play the game, so scalability of the system will be high.

Security: The other main goal of our game is protecting user data. So user's progress will be automatically saved in both local device and cloud. Cloud save will allow to load the player's progress when needed in particular cases such as re-installation, device issues etc.

Maintainability: Our game runs on mobile and tablet side. This system will be written in Unity with Android SDK tools and C#. The game will always updated and debugged according to the user's feedback and other new features.

Usability: Our game will be easy for user. The soft graphics and the specified color palette will be designed for not exhaust the user's eyes. A simple game menu (which will support multi-languages) and detailed walkthrough will make it easier for the user to understand game mechanics.

Availability: Our game will be available for more than 12000 different devices that supports android 4.0 or higher. Users don't have to connect to internet to play the game. Their in-game progress will be loaded in their google accounts when they connect the internet.
